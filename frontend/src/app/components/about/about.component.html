This is a project made for the course Natural Language Processing.

This application provides an user interface for text generation models created for 4 randomly chosen subreddits. These subreddits are cryptocurrency, me_irl, wallstreetbets and antiwork.

In addition to the mentioned models there is also a text classification model that classifies the generated output into 5 different categories. Each category corresponds to the specific subreddit, but the 5th category is other.

To use the application, just select a model from the list and in the Pilot view type the beginning of the text and press complete.
